Elastic Beanstalk CLI

Throughout this course, we used the Elastic Beanstalk web interface.

If you'd like, you can also use the EB CLI (Elastic Beanstalk Command Line Interface).


It is a very powerful tool that can hook into your Git repository. However, it is more complex to use and get used to. Therefore, I only recommend doing so once you feel comfortable.
