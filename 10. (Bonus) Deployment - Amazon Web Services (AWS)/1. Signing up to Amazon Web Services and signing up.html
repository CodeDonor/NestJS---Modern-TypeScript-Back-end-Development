In this section, I will guide you through the process of deploying an application to AWS using the Elastic Beanstalk service.

This requires you to have an Amazon Web Services account. During registration, you will have to provide payment details.

AWS Free Tier

Amazon Web Services offer a Free Tier for one year. During this one year, you are eligible for free usage of services (not all of them).

It's important that there are limitations to the free tier. Please make sure to read the Pricing section of each service when using AWS to be up-to-date with the latest Free Tier policies.

Read more on the AWS Free Tier


Free Tier and This Course

Throughout the course I only used services and configurations that fall under the free tier. However, I do not take responsibility for any costs that you might encounter due to incorrect AWS usage or deviation from the Free Tier.

It is your own responsibility to continuously check the AWS Free Tier quotas.


General Advice

  • If you use AWS for testing and education, make sure to turn off the services when you are done. This will help you prevent unwanted payments to AWS.

  • If you run services that are consumed by users, periodically check the AWS Billing Dashboard to keep an eye on your usage and costs.

  • You can use the Billing Dashboard to set up a budget as well as e-mail notifications for when the budget is forecasted to be exceeded.
